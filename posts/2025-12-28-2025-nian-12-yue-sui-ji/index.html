为了安装极简统计这个自己感觉非常适合的应用，在服务器上手动安装了 PHP ，这是很久很久以前安装 WordPress 时做过的事情了，远到那时候 Docker 还没出现。

因为旧的信息大概都是php配合nginx或者Apache之类的，自己现在不再用nginx，当年为了配置nginx而无数次的搜索并记下的笔记也早已变得毫无意义。现在用的是更便捷且自动配置ssl证书的Caddy，借助chatgpt或者deepseek编了份能用的Caddyfile配置文件，就立马可用了。

随着时间的推移，新的设备，新的工具出现，大批大批曾经耗费大量人力物力的东西，无论是实体物品还是代码数据都一一的被淘汰，消失的没有一点浪花。

搜索了一下 #Docker 是在 2013 年 3 月正式发布 0.1 版，也已经十几年了，让在 Linux 有了那种在 Windows 上使用绿色软件的爽快感觉，同时给无数人节省了无数的时间。

今天天气比较阴沉，时不时落下点小雨，但是气温还是维持在适宜的高位，总体还是舒适的体感。晚饭时间听了最新一期十字路口播客，讨论的人工智能问题。自己更倾向于将人工智能称为电子智能，因为它运行的基础是电子信号的动作，人创造了它的最初形态，但越往后来越是它自身的运行迭代和进化，所以人工智能已经不能准确的描述它了。

往远了讲，或许人类真的可能只是充当了一个引导程序，引导电子智能进入了硅基生命，然后人类迅速的跟不上前进的步伐而被远远的抛在后面。往近了讲，后续生活中的各个角落一定会被更多“ #电子智能 ”影响到而发生翻天覆地的重新组织。

一直在使用 Caddy，所以不论什么程序都不想再用 Nginx 了。

这两天理通了安装 Docker 版的 PHP 运行环境、Docker 版的 MySQL 或者 PostgreSQL 数据库，然后运行主机本地目录的 PHP 应用，通过 Caddy 转发出去。

按这种流程安装了极简统计以及 Typecho 和WordPress ，然后倒腾出很久很久前的 WP、 Typecho、Jekyll、Blogspot 的各种备份数据，通过脚本以及手工调节转换，终于全部导入到了新的 Wordpress 中。

挑了一大圈主题，最后还是回归到 WP 的“二〇一五”主题，时间都去哪了， 2015 距今也已经整整十年了。
